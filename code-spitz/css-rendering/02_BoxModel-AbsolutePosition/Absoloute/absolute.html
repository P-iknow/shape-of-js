Absolute

absolute는 offset parent 로 부터의 거리를 설정한다. absolute 요소에 top, bottom, left, right 값을 주지 않았을 때
기본 값은 static 이었을 때와 같다.
이것이 붉은 박스가 static 에 위치하고 있는 이유다.
left, top을 normal flow 로 그려지는 static 에 쓰면 무시된다. absolute 인 상태에서 float 속성을 주면 어떻게 될까? 무시된다. float는 normal flow 일 때만 성립한다. 왜? new BFC 를 만드는 것은 normal flow 일때 가능하다.
